# Event KeyCodes (Responsive)


A small webpage that displays keyboard event properties (`event.key`, `event.keyCode`, `event.code`) when you press keys. It now includes a responsive layout and a Key History panel that records recent key presses and modifiers.


## Features
- Shows `event.key`, `event.keyCode`, and `event.code`
- Highlights modifier keys (Shift, Ctrl, Alt, Meta)
- Keeps a rolling **Key History** (up to 50 entries)
- Buttons to **Clear** or **Copy** the history
- Responsive layout: works on mobile and desktop
- Keyboard accessibility and small HTML escaping utility


## Files
- `index.html` — markup and structure
- `style.css` — responsive styles
- `script.js` — logic, history, and UI updates
