Form Input Wave — README

A clean, responsive login form with an animated "wave" label effect and a Show/Hide password toggle. Built with plain HTML, CSS, and vanilla JavaScript — lightweight and easy to drop into any project.

Features

Animated label text where each character moves in a staggered "wave" when the input is focused or filled.

Show/Hide password toggle for better usability.

Responsive and minimal styling, easy to customize.

No dependencies (pure HTML/CSS/JS).
