 Sound Board — Interactive Audio Player

A simple, responsive sound board built using HTML, CSS, and JavaScript.
Each button plays a different sound, and the project includes extra features like volume control, random sound selection, and a global stop button.

This project is perfect for beginners learning DOM manipulation, audio elements, and responsive UI design.

Features
 Play Sound Buttons

Each button plays a unique sound (Applause, Boo, Gasp, Tada, Victory, Wrong).

Only one sound plays at a time — others stop automatically.

 Global Controls

Stop  — instantly stops every sound.

Random Sound — plays a random sound from the list.

Volume Slider — adjusts volume for all sounds together.

 Fully Responsive

Works smoothly on mobile, tablet, and desktop.

Buttons scale and rearrange automatically for smaller screens.

 Clean UI

Modern purple-gradient theme.

Smooth hover and click effects.

Card-styled layout for a premium feel.
